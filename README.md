# Node Draco Proxy
Proxy and UI in order to analyse trafic of Draconius GO using mitm.

## Install & Run

```
git clone https://github.com/dracoapi/nodedracoproxy.git
node ./bin/index.js
```
Then just configure your phone proxy to your computer IP, port 8888
